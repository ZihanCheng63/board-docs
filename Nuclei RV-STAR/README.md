# Nuclei RV-STAR

RV-STAR is an RISC-V evaluation development board based on the GD32VF103 MCU, providing onboard debugger, Reset and Wakeup user buttons, RGB LED, USB OTG, as well as EXMC, Arduino, and PMOD expansion interfaces and other resources.
